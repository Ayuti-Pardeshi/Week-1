# Part B: Basic Networking Concepts

## What is an IP Address?

An IP (Internet Protocol) Address is a unique address assigned to a device on a network. It helps devices identify and communicate with each other.

Example:
192.168.1.10

---

## What is a MAC Address?

A MAC (Media Access Control) Address is a unique hardware identifier assigned to a network interface card (NIC). It is used for communication within a local network.

Example:
00-1A-2B-3C-4D-5E

---

## What is a Default Gateway?

A Default Gateway is the device that forwards traffic from a local network to other networks, usually the internet. In most home networks, the router acts as the gateway.

---

## What is DNS?

DNS (Domain Name System) converts human-readable domain names such as google.com into IP addresses that computers can understand.

---

## Difference Between Public and Private IP Addresses

| Private IP                                | Public IP                    |
| ----------------------------------------- | ---------------------------- |
| Used inside local networks                | Used on the Internet         |
| Not accessible directly from the Internet | Accessible from the Internet |
| Assigned by router                        | Assigned by ISP              |
| Example: 192.168.1.10                     | Example: 49.x.x.x            |

## 3. What is the purpose of traceroute?

Traceroute helps identify the path that data packets take from a source device to a destination server. It shows each router or hop along the route and helps diagnose network issues.
